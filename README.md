## 实现的功能
+ 能够对表格进行增删查改
+ 点击复选框可以在数据统计图里出现线段
+ 在search框中可以进行对关键字的搜索
+ 点击search框下面的按钮可以对表格中的内容进行选择性显示
+ 能够对弹出的表单进行拖拽
+ 增加了填写表单时利用上下键进行选择，敲回车完成填写
